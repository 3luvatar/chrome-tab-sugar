h2. Browser events

 * On window open/close: a group is created/closed
 * On new tab: do nothing
 * On tab change: tab is created/updated
 * On tab closed: tab is closed
 * On tab reordered: tab is reordered
 * On browser closed: do nothing
 * On browser opened: restore session (optional)

h3. Resizing formulas

**tab:**
 width   tw
 height  th
 min width  TAB_MIN_WIDTH
 min_height TAB_MIN_HEIGHT
 scale      TAB_SCALE (=th/tw)
 nb tabs t

**group:**
 width   gw
 height  gh
 ratio   gr
 min width   GROUP_MIN_WIDTH
 min_height  GROUP_MIN_HEIGHT
 tab_spacing GROUP_TAB_SPACING_X
             GROUP_TAB_SPACING_Y
             GROUP_PADDING

**content:**
 columns col
 lines   lin

**count:**
 nb tabs per line    ntabx
 nb tabs per column  ntaby

**conditions:**
 **single sizes:**
  tw >= TAB_MIN_WIDTH
  th >= TAB_MIN_HEIGHT
  gw >= GROUP_MIN_WIDTH
  gh >= GROUP_MAX_HEIGHT
 **resizing:**
  ntabx * tw + (ntabx - 1) * GROUP_TAB_SPACING_X  = gw + 2 * GROUP_PADDING
  ntaby * th + (ntaby - 1) * GROUP_TAB_SPACING_Y <= gh + 2 * GROUP_PADDING
  **=>**
   tw  = [ gw + 2 * GROUP_PADDING - (ntabx - 1) * GROUP_TAB_SPACING_X ] / ntabx
   th <= [ gh + 2 * GROUP_PADDING - (ntaby - 1) * GROUP_TAB_SPACING_Y ] / ntaby
